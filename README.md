# 🚨 Aplicación de Emergencias 911

**Desarrollado por:** [TU MATRICULA] - [TU NOMBRE COMPLETO]

## 📱 Descripción

Aplicación móvil simple para el registro de eventos de emergencia del 911. Permite a los equipos de emergencia documentar sus intervenciones con fotos, descripciones y fechas.

## ✨ Características

- ✅ Registro de eventos con fecha, título, descripción y foto
- ✅ Almacenamiento local en el dispositivo (AsyncStorage)
- ✅ Lista de eventos registrados
- ✅ Vista detallada de cada evento
- ✅ Captura de fotos con la cámara o desde galería
- ✅ Interfaz simple y fácil de usar

## 🚀 Cómo Ejecutar

### Paso 1: Instalar dependencias
```bash
npm install
```

### Paso 2: Iniciar el proyecto
```bash
npm start
```

### Paso 3: Ejecutar en dispositivo
- Descarga **Expo Go** en tu celular (Play Store / App Store)
- Escanea el código QR que aparece en la terminal
- La app se abrirá automáticamente
